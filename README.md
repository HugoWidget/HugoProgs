# HugoProgs

## 项目介绍

帮助广大电教委对希沃功能进行增强与补充

## 环境依赖

Visual Studio 2022

## 功能

* 已发布

  1. HugoProgs命令行交互界面
  2. 希沃管家关闭与重启
  3. 希沃管家屏幕锁管理
  4. 希沃冰点服务交互
  5. 希沃冰点管理
  6. 希沃文件保护服务管理

* 已实现

  1. 希沃管家密码找回
  2. 希沃虚拟磁盘挂载

* 正编写

  1. 正在整合功能

* 将完成

  1. 暂无
