# HugoProgs

## 项目介绍

帮助广大电教委对希沃功能进行增强与补充

## 环境依赖

Visual Studio 2022

## 功能

* 已发布

  1. HugoProgs命令行交互界面
  2. 希沃管家关闭与重启
  3. 希沃管家屏幕锁管理
  4. 希沃冰点服务交互
  5. 希沃冰点管理
  6. 希沃文件保护服务管理
  7. 希沃虚拟磁盘挂载
  8. 希沃映像劫持
  9. 希沃下载与卸载
  10. 希沃管家密码找回
* 将完成
  1. 禁用希沃自更新
  2. 暂时禁用希沃网络连接

## 说明

Release中HugoProgs.zip为该项目完整编译产物，HugoProgs-bundle.zip还包含其他辅助工具
