# HugoPassword

## 项目简介
HugoPassword 是基于 C++ 开发的 Windows 平台希沃密码破解工具，核心用于**破解希沃管理密码/锁屏密码的密文**。工具支持手动输入密文和自动读取密文两种操作模式，可识别 V1/V2/V3 三种版本密文，破解结果按模式和类型排序输出，适用于快速解析希沃密码密文。

## 核心功能
- **双模式密文获取**：
  - 手动输入模式：手动指定密文版本（1/2/3）和密文内容，生成破解任务；
  - 自动读取模式：通过 AutoInfoAcquirer 自动从系统中读取密文、设备 ID、机器 ID 等信息，无需手动输入；
- **系统信息采集**
- **多版本解密支持**：内置 V1/V2/V3 三种版本解密器，适配不同版本希沃密码密文破解；
- **任务管理与执行**
- **可视化输出**：
  - 破解任务执行前展示完整任务信息；
  - 破解完成后按模式/类型排序输出结果；

## 使用方法
### 前置要求
- 运行环境：Windows 系统；
- 权限要求：**建议以管理员身份运行**（确保能读取系统注册表、公共应用数据目录下的 SeewoCore.ini 文件）；
- 文件依赖：自动读取模式需系统中存在 `%COMMON_APPDATA%\Seewo\SeewoCore\SeewoCore.ini` 文件（希沃客户端安装后生成）。

### 运行程序
- 直接双击运行或通过命令行启动程序，按照程序输入提示即可

## 注意事项
- 鉴于最新的希沃管家通常不将密文放在本地，本项目的自动模式可能失效
- PASSWORDV3仅支持本机计算，因为设备ID和机器ID对每个设备是唯一的
