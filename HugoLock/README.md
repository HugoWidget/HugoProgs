# HugoLock

## 项目简介
HugoLock 是基于 C++ 开发的 Windows 平台希沃管家窗口管理工具，核心用于**强制隐藏希沃管家置顶窗口并向 SeewoServiceAssistant.exe 进程注入自定义 DLL**。工具强制要求管理员权限运行，具备单实例保护机制，循环检测并隐藏置顶的“希沃管家”窗口，同时自动将 HugoHook.dll 注入到目标进程，适用于解除希沃管家窗口锁定、实现自动化窗口管控。

## 核心功能
- **单实例保护**：保证程序唯一运行，避免重复启动导致的窗口操作、进程注入冲突；
- **窗口检测与隐藏**
- **进程注入**：
  - 将同目录下的 `HugoHook.dll` 注入到所有 `SeewoServiceAssistant.exe` 进程；
- **无交互运行**：程序启动后无手动交互步骤，后台自动完成窗口检测、隐藏、进程注入全流程。

## 使用方法
### 前置要求
- 运行环境：Windows 系统；
- 权限要求：**必须以管理员身份运行**；
- 进程依赖：注入功能仅针对 `SeewoServiceAssistant.exe` 进程，无此进程则注入步骤自动跳过。

### 运行程序
1. 确认程序同目录存在 `HugoHook.dll`；
2. 右键点击 `HugoLock.exe`，选择「以管理员身份运行」；

## 注意事项
- 权限相关：
  - 非管理员权限运行会直接退出，无法执行窗口隐藏、进程注入操作；
  - 调试权限提升失败会导致进程注入失败，但不影响窗口隐藏功能；
- 文件依赖：
  - 程序同目录缺失 `HugoHook.dll` 时，进程注入步骤会失败；
- 终止方式：程序无退出指令，需手动关闭控制台或结束进程，强制终止不会导致系统异常。
