# HugoScreenSaver

**HugoScreenSaver** 是一个自动解除“希沃管家”屏保的小工具。它通过持续检测全屏置顶的“希沃管家”窗口，模拟屏幕靠左位置（距左边缘 80 像素）的鼠标点击来尝试解除屏保。

## 功能简介

- 自动监控名为“希沃管家”的全屏置顶窗口
- 检测到目标窗口后，等待 800 毫秒稳定，然后点击屏幕
- 支持两种运行模式：
  - **默认模式**：连续 5 次点击后若窗口仍然存在，程序自动退出
  - **保持模式**：使用 `keep` 参数启动，程序将一直保持运行，反复尝试解除屏保
- 单实例运行，避免重复启动

## 使用方法

### 命令行参数

```shell
HugoScreenSaver.exe [keep]
```

### 运行行为

1. 程序启动时会申请管理员权限。
2. 检测当前是否已有实例运行，若有则直接退出。
3. 进入监控循环，每 500 毫秒检查一次是否存在名为“希沃管家”且置顶的窗口。
4. 发现窗口后，等待 800 毫秒让窗口稳定，然后模拟鼠标移动到屏幕靠左位置并执行左键点击。
5. 根据运行模式决定后续行为（退出或继续监控）。

### 注意事项

- 在锁屏界面下，单纯的点击并不会让窗口消失，为避免干扰正常使用，此时程序会连续尝试后按规则退出。

> 如果有可以区分锁屏界面和屏保界面的方法，欢迎告知

- 若使用 `keep` 模式，程序会在后台持续运行，可在系统托盘或任务管理器中结束进程。
