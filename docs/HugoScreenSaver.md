# HugoScreenSaver

**HugoScreenSaver** 是一个自动解除“希沃管家”屏保的小工具。它通过持续检测全屏置顶的“希沃管家”窗口，模拟屏幕靠左位置（距左边缘 80 像素）的鼠标点击来尝试解除屏保。

## 功能简介

- 自动监控名为“希沃管家”的全屏置顶窗口
- 检测到目标窗口后，等待 800 毫秒稳定，然后点击屏幕
- 支持两种运行模式：
  - **默认模式**（无参数）：连续 5 次点击同一窗口仍未消失时，该窗口会被加入忽略列表，之后不再处理，程序继续监控其他可能出现的屏保窗口
  - **保持模式**（使用 `keep` 参数）：程序将一直保持运行，反复尝试点击任何检测到的“希沃管家”窗口，不使用忽略列表
- 单实例运行，避免重复启动

## 使用方法

### 命令行参数

```shell
HugoScreenSaver.exe [keep]
```

### 运行行为

1. 程序启动时会申请管理员权限。
2. 检测当前是否已有实例运行，若有则直接退出。
3. 进入监控循环，每 500 毫秒检查一次是否存在名为“希沃管家”且置顶的窗口。
4. 发现窗口后，等待 800 毫秒让窗口稳定，然后模拟鼠标移动到屏幕靠左位置并执行左键点击。
5. 根据运行模式决定后续行为：
   - 无 `keep` 参数：若同一窗口连续 5 次点击后仍然存在，将其加入忽略列表，后续跳过该窗口，程序继续运行
   - 有 `keep` 参数：始终尝试点击，不加入忽略列表

### 注意事项

- 在锁屏界面下，单纯的点击并不会让窗口消失。

> 如果有可以区分锁屏界面和屏保界面的方法，欢迎告知

- 若使用 `keep` 模式，程序会在后台持续运行，可在系统托盘或任务管理器中结束进程。
