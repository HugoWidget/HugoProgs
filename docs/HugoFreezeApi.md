# HugoFreezeApi

## 项目简介
HugoFreezeApi 是基于 C++ 开发的 Windows 平台工具，核心用于**通过调用指定 API 交互管理磁盘冻结状态**。工具提供可视化交互式菜单，支持磁盘冻结状态的设置、查询、尝试冻结操作，操作结果实时反馈，适用于便捷管理磁盘冻结相关的 API 调用操作。

## 核心功能
- **API 交互能力**：调用指定 API 接口实现磁盘冻结管理
- **交互式菜单操作**：提供可视化菜单，支持选择设置/查询/尝试冻结/退出操作，流程直观；
- **日志与控制台管理**：内置日志系统输出操作结果和错误信息，控制台本地化配置，提升使用体验；
- **配置自动读取**：运行时自动读取目标 API 的 IP 和端口配置，并显示在交互菜单中。

## 使用方法
### 前置要求
- 运行环境：Windows 系统；

### 1. 启动程序
直接运行编译后的可执行文件 `HugoFreezeApi.exe`，程序会自动初始化控制台、日志系统，并加载 API 配置（IP/端口）。

### 2. 交互操作流程
程序启动后会显示交互菜单，示例如下：

目标 IP: [默认 IP]:[端口]请选择操作：

- 1.设置冻结磁盘
- 2.查询冻结磁盘
- 3.尝试冻结磁盘
- 4.退出程序
- 请输入选择：

#### 操作1：设置冻结磁盘
- 输入数字 `1` 并回车；
- 根据提示输入**磁盘标识字符串**；
- 程序调用接口，控制台输出设置结果。

#### 操作2：查询冻结磁盘
- 输入数字 `2` 并回车；
- 程序调用接口，控制台输出查询到的磁盘冻结状态信息。

#### 操作3：尝试冻结磁盘
- 输入数字 `3` 并回车；
- 根据提示输入**磁盘标识字符串**；
- 程序调用接口，控制台输出尝试冻结的结果。

#### 操作0：退出程序
- 输入数字 `0` 并回车，程序立即退出。

## 注意事项
- 每次冰冻必须包含C盘，否则会冻结失败
